# FET312 Derin Öğrenme Projesi: Deepfake Tespiti
Bu repo, 2025-2026 Güz Dönemi FET312 Derin Öğrenme dersi kapsamında geliştirilen proje dosyalarını içermektedir.
Proje Başlığı: Derin Evrişimli Sinir Ağları ile Deepfake Video Tespiti
Ekip Üyeleri:
   Khaiitmurod Khabibullayev
   Abdumajid Abdulkhaev
Bu projede, FaceForensics++ veri seti kullanılarak deepfake videolarını tespit eden modeller geliştirilmiştir. Vize aşaması için her grup üyesi, farklı temel CNN mimarileri tasarlamış ve performanslarını karşılaştırmıştır.
    FET312_22040101116_Detectify_1.ipynb: Khaiitmurod Khabibulllayev tarafından geliştirilen 3 katmanlı CNN modeli.
    FET312_6454545_Detectify_1.ipynb: Abdumajid Abdulkhaev tarafından geliştirilen ... modeli.
    FET312_5484845_Detectify_1.ipynb: Muhammet tarafından geliştirilen ... modeli.
Kodlar Google Colaboratory ortamında geliştirilmiştir. Notebook'u Colab'da açtıktan sonra, gerekli veri setinin Google Drive'da belirtilen yolda olduğundan emin olup hücreleri sırasıyla çalıştırmak yeterlidir.
